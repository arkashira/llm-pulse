# Partner Targets

## Integration Roadmap

### Phase 1: Immediate (0-3 months)
1. **LangChain**
   - **Rationale**: LangChain is widely used for building applications with LLMs. Integrating with LangChain will allow users to monitor their LLM applications seamlessly.
   - **Free-Tier Limits**: 10,000 API calls/month
   - **Integration Effort**: S
   - **Value-Add**: Solves the job of monitoring LLM performance within existing LangChain applications.

2. **Hugging Face**
   - **Rationale**: Hugging Face is a popular platform for accessing a wide range of LLMs. Integrating with Hugging Face will provide users with performance insights for various models.
